# ML Basics

This repository contains fundamental machine learning and data analysis concepts implemented using Python.

## Contents
- NumPy basics for numerical computing
- Pandas for data manipulation
- Data visualization using Matplotlib and Seaborn

## How to Run
1. Clone the repository
2. Install dependencies using:
   pip install -r requirements.txt
3. Open notebooks using:
   jupyter notebook

## Goal
Build a strong foundation for advanced machine learning and open-source AI projects.
